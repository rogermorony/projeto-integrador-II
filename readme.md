Este projeto faz parte da disciplina de <b>Projeto Integrador II</b>.  
O objetivo é evoluir o front-end de um e-commerce fictício da <b>Serra Verde Gramas</b>, adicionando <b>interatividade com JavaScript</b>, manipulação da DOM e validação de formulários.

---

## 🚀 Funcionalidades Implementadas

- Adicionar produto ao carrinho (simulação visual)
- Alteração dinâmica de botão (feedback ao usuário)
- Botão de copiar link da página
- Compartilhamento de produto via WhatsApp
- Sistema de favoritar produto
- Validação completa do formulário de contato
- Mensagens de erro dinâmicas
- Alternância de tema (Light/Dark Mode)

---

## 📌 Páginas do Projeto

- **Página Inicial (`index.html`)**
  - Banner principal
  - Vitrine de produtos
  - Sidebar com categorias

- **Página de Detalhes do Produto**
  - Nome, imagem e preço
  - Botão “Adicionar ao Carrinho”
  - Botões de ação (copiar, compartilhar, favoritar)

- **Página de Contato (`contato.html`)**
  - Formulário com validação em JavaScript
  - Feedback de erro e sucesso

---

## 🎨 Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Google Fonts** (Plus Jakarta Sans / Inter)
- **Ionicons**

---

## 🧠 Conceitos Aplicados

- Seleção de elementos da DOM (`querySelector`, `getElementById`)
- Eventos (`click`, `input`, `submit`)
- Manipulação de conteúdo (`innerText`, `value`)
- Manipulação de atributos (`setAttribute`)
- Manipulação de classes (`classList`)
- Validação de formulário
- Alteração dinâmica de interface
